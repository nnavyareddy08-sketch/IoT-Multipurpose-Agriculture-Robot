# IoT Enabled Multipurpose Agriculture Robot

## Project Overview
This project is an IoT-enabled agriculture robot designed for crop monitoring and field operations. The robot uses Arduino Mega and ESP8266 (NodeMCU) to enable remote control through Adafruit IO over Wi-Fi.

## Features
- Remote robot movement
- Servo motor control
- Relay-based irrigation and spraying
- Wireless communication using ESP8266
- Cloud-based control using Adafruit IO

## Hardware Used
- Arduino Mega
- ESP8266 (NodeMCU)
- DC Motors
- Servo Motor
- Relay Module
- Motor Driver

## Software Used
- Arduino IDE
- C++
- Adafruit IO
- ESP8266 Library
